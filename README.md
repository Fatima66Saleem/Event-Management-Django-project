# 📅 Event Management System (Django)

The complete Django project is available for download at the link below (due to GitHub file size limits).

---

## 📂 How to Use:
1. **Download:** Click the link above and download the ZIP file.
2. **Extract:** Unzip the folder on your computer.
3. **Run Server:** Open terminal in the folder and type:
   `python manage.py runserver`
4. **View Website:** Go to `http://127.0.0.1:8000/`
